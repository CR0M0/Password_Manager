KeyFort Password Manager - Quick Start Guide

* Prerequisites:
- Node.js v16.0.0 or higher
- Firebase account with project created (enable Authentication and Firestore)

* Installation & Setup
- Install Dependencies: Navigate to both "password-manager" and "server" directories and run npm install
- Start Backend Server: Open terminal, navigate to server directory "server", and run npm run dev (server runs on port 5000)
- Start Frontend: Open new terminal, navigate to client directory "password-manager", and run npm run dev (app runs on port 5173)
- Access Application: Open browser and navigate to http://localhost:5173
- Register Account: Create account with username, email, and master password
- Save Secret Phrase: Write down the 12-word recovery phrase displayed (shown only once!)
- Start Managing Passwords: Create folders and add password entries to your secure vault
